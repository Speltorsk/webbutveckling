How to Use Flexbox

Flexbox is a powerful CSS layout model that makes it easier to design flexible and responsive layouts. To use flexbox, you first set display: flex on the parent container. This turns its children into flex items. The main axis runs in the direction specified by flex-direction (by default, row), and the cross axis runs perpendicular to it.

Common flexbox container properties include justify-content (which aligns items along the main axis), align-items (which aligns items along the cross axis), and flex-wrap (which controls whether items wrap onto new lines).

For individual flex items, properties like flex-grow, flex-shrink, and flex-basis allow you to control how the items grow or shrink relative to their siblings. In many cases, setting flex: 1 on all items creates equal distribution.
